# Pizza-Sales-Analysis

This project focuses on analyzing pizza sales data to uncover business insights that can support operational efficiency, revenue growth, customer preference analysis, and sales optimization. The analysis was performed using Python and data analysis libraries to explore sales performance across pizza categories, sizes, order patterns, and customer purchasing behavior. The objective was to identify key trends, sales drivers, peak business periods, and product performance indicators that can help improve business decision-making and strategic planning.

## Skills and Tools Covered:
* Exploratory Data Analysis (EDA): Data cleaning, summary statistics, feature distributions, trend analysis, correlation analysis
* Data Preprocessing: Handling missing values, data type validation, feature transformation, date and time analysis
* Sales Analysis: Revenue analysis, order trend analysis, category and size performance evaluation
* Business Intelligence & Insights: Identification of peak sales periods, top-performing pizzas, and customer ordering patterns
* Data Visualization: Bar charts, histograms, count plots, heatmaps, line charts
* Statistical Analysis: Distribution analysis, outlier analysis, univariate and bivariate analysis
* Tools & Libraries: Python, Pandas, NumPy, Matplotlib, Seaborn
* Jupyter Notebook: End-to-end analytical workflow and reporting

##My Learning

This project involved performing an end-to-end sales analysis on pizza order data to derive actionable business insights and improve understanding of sales performance trends. The analysis began with Exploratory Data Analysis (EDA), which included data cleaning, validation of data quality, handling missing values, and understanding the structure and distribution of the dataset.

The project explored key business metrics such as total revenue, order volume, pizza category performance, pizza size preferences, and sales trends across different time periods. Date and time-based analysis was performed to identify peak ordering hours, busiest days, and seasonal purchasing patterns. The analysis also evaluated the contribution of different pizza categories and sizes to overall revenue generation.

Visualization techniques were used extensively to uncover trends and communicate insights effectively. Various plots and charts were created to analyze customer ordering behavior, top-selling products, and operational performance patterns. Outlier analysis was also performed to validate unusual sales values and distinguish between genuine business scenarios and data anomalies.

The project demonstrated how data analytics can be used to support business decision-making by identifying revenue drivers, customer preferences, and operational trends. It also strengthened practical skills in data preprocessing, exploratory analysis, visualization, and insight generation using Python-based analytics tools.
